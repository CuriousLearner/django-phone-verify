Welcome to Django Phone Verify's documentation!
===============================================

``Django Phone Verify`` is a simple Django app that allows you to easily add phone number verification to your project using the security code sent via SMS.

The app uses the Twilio/Nexmo API to send verification codes via SMS and allows users to verify their phone numbers by entering the code they receive.
